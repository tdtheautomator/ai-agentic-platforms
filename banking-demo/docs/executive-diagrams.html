Agent Interactions

Session 1: Write rules → Semantic; Seed context → In-Context
Session 2: Read context → In-Context; Cache results → Working
Session 3: Query rules → Semantic; Cache risk level → Working
Session 4: Read risk level → Working; Log checks → Episodic
Session 5: Read all stores; Write decision → All 4 stores
